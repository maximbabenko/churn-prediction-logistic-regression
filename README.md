## Churn Prediction — Logistic Regression

В данном проекте решается задача прогнозирования оттока клиентов банка с использованием логистической регрессии.

Основной акцент сделан не только на построении модели, но и на анализе её поведения при различных настройках: регуляризации, дисбалансе классов и выборе порога классификации.


### Цель проекта

Построить интерпретируемую модель логистической регрессии для прогнозирования оттока клиентов и провести сравнительный анализ различных конфигураций модели с целью оценки их влияния на качество предсказаний
